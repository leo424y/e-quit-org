雅加達環球報：印尼禁止在火車上吸菸
（Smoking Ban on Trains Now in Force - Jakarta Globe） / March 5, 2012
 
印尼在轉型的柴電機車（火車）運作數十年後，自週四開始全面實施鐵路運輸禁菸規定，終結旅客在火車車廂吸菸的時代。印尼鐵路總公司Kereta Api Indonesia的普禾格多代言人Surono表示，鐵路公司將嚴格實施火車禁菸，在車上吸菸的乘客將在下一站被請下車，而禁菸規定上路的第一天，乘客都還算遵守規定，有少數乘客會下車至各車站設置的吸菸區吸菸，且多數乘客已知悉政府的禁菸規定。位宣導禁菸，車站已懸掛禁菸之宣導布條與海報，甚至印在車票上。在車廂張貼宣導標示，公司員工發現約80%的宣導貼紙會被乘客損壞或移除，但鐵路公司還是持續重新更新。
 
在西爪哇Bogor站的經理Rochman表示，在實施禁菸規定的初期，車站的工作人員會先禮貌地提醒吸菸者把菸熄掉，而中爪哇車站除在乘客買車票的車站南北入口懸掛布條，並在站內分發2,000份傳單。實施禁菸規定的第一天有30名乘客被警告，車站也要求小販不要在車廂內販賣菸品等措施。
